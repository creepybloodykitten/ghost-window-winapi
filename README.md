# ghost-window
lightweight utility for Windows that allows you to keep a window always on top and adjust its transparency

![2026-01-17 14-03-13 (online-video-cutter com) (1)](https://github.com/user-attachments/assets/f3ebbdca-4d1d-480b-99b8-2a2281b7a001)

## usage
add the application to startup (as an administrator so that the utility works with all windows)

ctrl+shift+a to switch between normal and always on top(transparency/ghost) mode

when it in second mode -> right alt + mouse wheel to edit transparency of the window

## consumption
500 kb always in the background

0% CPU always
